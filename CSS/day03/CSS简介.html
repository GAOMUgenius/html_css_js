CSS（层叠样式表）

层叠样式表(Cascading style sheets，缩写为 CS5)，是一种 样式表 语言，用来描述 HTML或XML(包括如 SVG、MathML. XHTML 之类的 XML 分支语言)文档的呈现。CSs 描述了在屏幕、纸质、音频等其它媒体上的元素应该如何被澶染的问题。

CSS 是开放网络的核心语言之一，由 W3C,规范实现诗浏览器的标准化。(SS 节省了大量的工作。 样式可以通过定义保存在外部,c5s 文件中，同时控制多个网页的布局，这意味着开发者不必经历在所有网页上编辐布局的麻烦。 CSS 被分为不同等级:CSS1现已废弃，(SS2.1是推特标准，(553 分成多个小模块目正在标准化中。
